## Installing slurmtools

`slurmtools` can be installed with `pak`: 

```R
#install.packages("pak")
pak::pkg_install("a2-ai/slurmtools")
```

upon loading slurmtools you should receieve an onAttach message alerting you to several options that are used with `slurmtools`.
```R
library(slurmtools)
```
<Card>
